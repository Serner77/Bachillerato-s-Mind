# Bachillerato-s-Mind

## USO:

Para ejecutar la app hay que:

- 1: Clonar el repo con:

      git clone https://github.com/Serner77/Bachillerato-s-Mind
  
- 2: Acceder desde consola al directorio static y dentro de este directorio ejecutar un servidor http en python con:

       python3 -m http.server 8000

- 3: Acceder mediante un navegador a:
  
      localhost:8000
